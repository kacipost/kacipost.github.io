Pacmen Exercise





The PacMan Exercise was a project completed during the MIT xPro Full-Stack Engineering with MERN Professional Certification Course. In this project the main learning component was learning how to manipulate the DOM. It uses JavaScript fnctionality to make a PacMan factory so you can add as many PacMen to the page as you'd like by clcking the "Add Pacman" button and the "Start Game" button makes the PacMen move in random directions as random velocities. They then bounce off the screen edges to stay within the screen size.
